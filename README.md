A simple and interactive To-Do List web application built with HTML, CSS, and JavaScript. 
Users can add tasks, mark them as completed, delete tasks, and all tasks are saved in localStorage,
so they persist even after refreshing the page.

  Features

- Add new tasks

- Mark tasks as completed (with a line-through)

- Delete tasks

- Tasks are saved in local storage and reload on page refresh

- Responsive and clean UI

Technologies Used

- HTML

- CSS

- JavaScript

- Font Awesome (for icons)
